# AI UI Design Prompt for Property Management App

## Concise Prompt (Copy-Paste Ready)

```
Design a property management web application UI called "CasaTrack" for tracking home purchases.

**Pages & Structure:**

1. **Login Page**
   - Centered glassmorphism card with password form
   - Logo and app title

2. **Main Dashboard** (three view modes):
   - **Cards View**: Grid layout with property cards
   - **Kanban Board**: 7-column workflow board (Seen → Interested → Contacted → Visited → On Hold → Irrelevant → Purchased)
   - **Map View**: Google Maps with property markers

3. **Property Detail Modal**
   - Property information, images, notes, attachments
   - Status management and flagging

4. **Add/Edit Property Form**
   - Modal for creating/editing properties

**Design Style:**
- Modern, minimalist, clean
- Mobile-responsive

**Key Features:**
- Property status tracking
- Notes and attachments
- Map visualization
- Flagging system
- Search and filtering
```
